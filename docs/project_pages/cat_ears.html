Cat Ears

Home

My friend always wanted a pair of cat ears. They're great for all sorts of occasions -- halloween, formal dining, research conferences... I also figured it'd be a fun project over winter break so here we are.

Design Ideas

I was going for an industrial cyberpunk aesthetic here, reminiscent of Akira or Ghost in the Shell. The ears are intentionally asymmetric to give the impression of a hasty construction. The various color pallets were also selected to invoke this chaotic atmosphere.

Construction

The guts are very similar to that of the bird lamp. Inside is a small neopixel ring, a three led line, and two individual neopixels for backlighting. An arduino nano is used for control and Molex Picoblades were used for easy reconfiguration. A small button allows the user to toggle between control modes. The two ears and electronics holder were modeled in Fusion 360 and 3D printed in carbon fiber filled nylon.

Programming

I made a pretty basic arduino script to run the lights. It takes input from a button and allows the user to toggle between a spinning mode, pure colors, rainbow, and a color picker. The color picker allows a user to send a rgb color code over serial to the ears. This is then displayed with the built in neopixels. While it probably would have been easier to just choose colors with guess and check, I thought it'd be fun to be able to update colors live if connected to the serial port.
